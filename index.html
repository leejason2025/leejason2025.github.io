Computer Science Portfolio

Here are reports on the projects I have been working on in my CS 180 class on Computer Vision and Computational Photography.

Project 1: Colorizing Images of the Russian Empire

This project explores colorizing grayscale images using alignment techniques. Key Concepts: Normalized Cross Correlation, Exhuastive Align, Pyramid Align, Guassian Pyramids, and Histogram Equaliziation.

Project 2: Fun with Filters and Frequencies

This project focuses on applying image filters and understanding frequency domains. Key Concepts: Guassian Blur, Fourier Transform, Guassianstacks, and Laplacian Stacks.

Project 3: Face Morphing and Modeling a Photo Collection

In this project, facial morphing techniques are used to create smooth transitions. Key Concepts: Deluaney Triangulation and Affine Transformations.

Project 4: Auto-stitching and Photo Mosaics

This project involves generating panoramas by stitching together images. Key Concepts: Homgraphies, Harris Corner Detection, Adaptive Non-Maximal Suppression, Feature Descriptor Extraction, and RANSAC.
